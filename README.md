# GSEA - Esqueleto opción B (DEMO)

Proyecto base semi-implementado para el reto final de Sistemas Operativos.
Incluye estructura, parser de argumentos, manejo básico de archivos,
concurrencia de ejemplo y algoritmos de demostración para compresión y
encriptación. Los algoritmos deben ser revisados, modificados y justificados
por el estudiante según el enunciado oficial.
